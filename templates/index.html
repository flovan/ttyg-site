TTYG — Talk To Your Grid v0.1.0

TTYG is an attempt at creating a smart SCSS grid mixin that understands human language, while at the same time outputting DRY code. For optimal results, this mixin should be used in combination with CSS3 level selectors, as well as a CSS minifier and/or mixin merger.

Columns are on-demand, and no CSS is rendered until you ask for it through an @include command. It is also possible to make ready-to-use classes available if that's what floats your boat.

Project configuration

If you want to configure TTYG on a per-project basis, you can call the following mixin and pass in your settings:

@include ttyg-config(
   context: 12,
   gutter: 5%,
   breakpoint: 650px,
   grid-width: 1140px,
   page-padding: 4%,
   font-size: 16px
);

The settings above are the defaults, and settings are used throughout useage of the mixin, usually as fallback if no specific values are used.

Offset

Usage of offset (followed by an amount of columns) will push columns to the left.

Centering

Good ol' margin: 0 auto; centering.

@include col([number] centered);
4 centered
8 centered

Non-breaking

Usage of no-break will not add any min-width media queries and thus columns will persist on smaller screens.

@include col([number] no-break);
6 no-break

Context

By using of, a different context can be used for calculating column widths.

@include col([number] of [number]);
3 of 5
2 of 5

Reassembly Point and Range

Start or end a layout on a specific point or assign it to a range by using from and to followed by a pixel value.
The latter one of the includes below would use the config breakpoint as from value.

@include col([number] from [pixel value];
@include col([number] from [pixel value] to [pixel value]);
@include col([number] to [pixel value]);
6
Default: from 650px
6 from 500px
6 to 800px
Default: from 650px
6 from 800px to 1000px
